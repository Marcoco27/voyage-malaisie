## Format de nommage

```
[CODE]-[NUMERO].jpg

Exemples:
KL01-001.jpg, KL01-002.jpg, KL01-003.jpg
CH01-001.jpg, CH01-002.jpg
TN01-001.jpg
```

## Sur Cloudinary

Créer un dossier par code :
- `KL01/`
- `CH01/`
- `TN01/`
- `KB01/`
- `PE01/`
- `PE02/`
- `CH02/`
- `TI01/`
- `SG01/`
- `ML01/`
- `KS01/`
- `KL02/`

Uploader les photos dans le dossier correspondant avec le nom `[CODE]-[NUMERO].jpg`
